# Plataforma Digital Raizes e Saberes

Showroom comercial estatico do Ecossistema Educacional Raizes e Saberes.

## Estrutura publicada

- `index.html`: Home comercial.
- `biblioteca.html`: Biblioteca Digital Premium.
- `styles.css`: estilos globais.
- `script.js`: interacoes da Home e Biblioteca.
- `assets/`: imagens oficiais e mockups usados nas telas.
- `docs/`: memoria e organizacao do projeto.

## Deploy na Vercel

Este projeto pode ser publicado como site estatico, sem etapa de build.

Configuracao recomendada na Vercel:

- Framework Preset: `Other`.
- Build Command: vazio.
- Output Directory: vazio ou raiz do projeto.
- Install Command: vazio.

Quando o repositorio GitHub estiver conectado a Vercel, cada push na branch de producao dispara uma nova publicacao.
